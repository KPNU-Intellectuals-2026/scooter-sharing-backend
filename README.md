# 🧠 Інтелектуальна адаптація вебінтерфейсів

![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)
![Language: JavaScript](https://img.shields.io/badge/language-JavaScript-F7DF1E?logo=javascript&logoColor=black)
![Version](https://img.shields.io/badge/version-1.0.0-blue)
![Platform: Web](https://img.shields.io/badge/platform-Web-brightgreen)

Система для динамічної адаптації інтерфейсів на основі поведінкових даних користувача. Проєкт спрямований на покращення доступності та персоналізації вебдосвіду.

## 🚀 Технологічний стек

* **JavaScript:**
    * Стандарт для роботи з DOM та адаптації інтерфейсу у браузері.
    * Збір поведінкових даних без додаткового ПЗ.
    * Використання бібліотек для аналізу дій користувача.
* **HTML5 та CSS3:**
    * Створення доступних та семантичних елементів.
    * Зміна контрасту, розміру тексту та відступів у реальному часі.
* **Додаткові інструменти:**
    * **ARIA:** підвищення доступності компонентів.
    * **TensorFlow.js / ML5.js:** адаптація на основі машинного навчання.
    * **Chart.js:** візуалізація зібраної статистики.
    * **Node.js:** логування та збереження даних.
